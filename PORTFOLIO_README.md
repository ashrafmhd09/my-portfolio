# Interactive Timeline Portfolio

A story-driven portfolio website showcasing Ashraf Mohammad's 12+ year journey in software development.

## Features

### 🎯 Design Philosophy
- **Timeline-based narrative** - Career journey presented as an interactive story
- **Pure CSS layouts** - Built exclusively with CSS Grid and Flexbox (no libraries)
- **Performance optimized** - Vanilla JavaScript with debouncing and Intersection Observer
- **Fully responsive** - Mobile-first design that works on all devices

### ✨ Interactive Elements

1. **Scroll Progress Bar** - Visual indicator of reading progress
2. **Animated Statistics** - Counter animations for key metrics
3. **Timeline Reveal** - Cards fade in as you scroll
4. **Hover Effects** - Interactive cards with smooth transitions
5. **Tech Tag Interactions** - Clickable technology badges with ripple effects
6. **Parallax Hero** - Subtle background movement on scroll
7. **Mobile Navigation** - Hamburger menu for small screens
8. **Easter Egg** - Try the Konami Code! (↑ ↑ ↓ ↓ ← → ← → B A)

### 📱 Sections

1. **Hero** - Introduction with animated stats
2. **Story Intro** - Context-setting introduction
3. **Timeline Journey** - 9 career milestones from 2006 to present
   - Education (2006-2010)
   - Trainee at CMC LTD (2012)
   - IT Engineer at CMC LTD (2012-2014)
   - Martha Stewart Living (2014)
   - Digital Pulp (2015)
   - Sogeti (2015-2017)
   - Arxscan (2017-2021)
   - DS Federal (2022-2025) - Featured "Most Inspirational Employee" award
   - Veolia (2025-Present) - Current position
4. **Technical Arsenal** - 8 skill categories
5. **Contact** - Easy ways to get in touch

### 🎨 Design Highlights

- **Dark theme** with gradient accents
- **Smooth animations** with CSS transitions
- **Professional color scheme** - Blues, purples, and accent colors
- **Typography** - Inter for text, Fira Code for technical elements
- **Visual hierarchy** - Clear information architecture

### 🚀 Technology Stack

**Built with:**

**Features:**

### 🎯 Performance Features

- **Debounced scroll events** - Optimized for 60fps
- **Lazy loading ready** - IntersectionObserver for images
- **Request Animation Frame** - Smooth scroll animations
- **CSS-only animations** - Hardware-accelerated transforms
- **No external dependencies** - Fast load times

### 🌟 Unique Features

1. **Story-driven narrative** - Each role tells a story
2. **Visual timeline** - Alternating left/right layout
3. **Achievement highlights** - Key accomplishments for each role
4. **Technology evolution** - Shows tech stack growth over time
5. **Award recognition** - Highlights "Most Inspirational Employee" badge
6. **Current role indicator** - Special styling for present position
